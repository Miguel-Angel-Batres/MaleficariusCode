Competitive Programming as a tool for improvement

As is known, programming has many aspects to address. As an engineer, I can say that, even if programming standards are known for developing any type of applications, solving daily problems is truly beneficial.

Here are some web sites that helps you find hard problems that keep your mind busy:

  • CodeChef: A competitive programming platform that offers monthly challenges and regular contests. https://www.codechef.com/
  • AtCoder: A Japanese-origin platform with interesting problems and regular competitions. https://atcoder.jp/
  • LeetCode: While LeetCode primarily focuses on technical interviews, it also provides competitive programming problems. https://leetcode.com/
  • HackerRank: It offers a variety of programming challenges, not only competitive but also for skill development in various areas. https://www.hackerrank.com/
  • TopCoder: One of the oldest platforms for programming competitions. It offers both competitive programming contests and development challenges. https://www.topcoder.com/
  • Sphere Online Judge (SPOJ): A platform with a vast collection of problems and competitions. https://www.spoj.com/
