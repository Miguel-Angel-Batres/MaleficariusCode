Zacatecas Project

The Zacatecas Joint Project is a web development initiative aimed at promoting a musical group in the state of Zacatecas, Mexico. The project is still in development and is expected to be ready before 2025.

Retro Death

Retro Death is a game made in Godot with graphics similar to PSP games and gameplay inspired by 90s first-person shooters. The game is still in development and is expected to be ready before 2025.

Phantom Beasts

Phantom Beasts is a horror and skill-based game made in Godot with simple mobile-friendly graphics. The game was developed by me in 2023 and, although it is not yet public, it serves as a good project for learning to program in Godot.

The game involves dodging ghosts that appear on the screen by moving the character across the map, trying to last as long as possible. The longer you last, the faster the ghosts move. Additionally, the game saves your high score and allows you to challenge others to beat it.

Guitar Meme

A video game made in Java that tries to mimic Guitar Hero, including both videos and music in a parody style, featuring uncommon characters. It has a very simple design and gameplay very similar to Guitar Hero.

This project was developed by me in 2023, and although it is not yet public, it serves as a good project to learn programming in Java.
