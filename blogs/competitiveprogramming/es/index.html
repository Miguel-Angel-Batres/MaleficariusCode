Programación Competitiva


La programación competitiva como herramienta de mejora

Como se sabe, la programación tiene muchos aspectos para abordar. Como ingeniero, puedo decir que, aunque se conozcan estándares de programación para desarrollar cualquier tipo de aplicaciones, resolver problemas diarios es realmente beneficioso.

Aquí hay algunos sitios web que te ayudarán a encontrar problemas difíciles que mantendrán tu mente ocupada:

  • CodeChef: Una plataforma de programación competitiva que ofrece desafíos mensuales y concursos regulares. https://www.codechef.com/
  • AtCoder: Una plataforma de origen japonés con problemas interesantes y competiciones regulares. https://atcoder.jp/
  • LeetCode: Aunque LeetCode se centra principalmente en entrevistas técnicas, también ofrece problemas de programación competitiva. https://leetcode.com/
  • HackerRank: Ofrece una variedad de desafíos de programación, no solo competitivos, sino también para el desarrollo de habilidades en diversas áreas. https://www.hackerrank.com/
  • TopCoder: Una de las plataformas más antiguas para competiciones de programación. Ofrece tanto concursos de programación competitiva como desafíos de desarrollo. https://www.topcoder.com/
  • Sphere Online Judge (SPOJ): Una plataforma con una vasta colección de problemas y competiciones. https://www.spoj.com/

Introducción


¿Qué es la programación competitiva?

La programación competitiva es un deporte mental que generalmente se realiza a través de Internet o una red local, donde los participantes intentan programar según las especificaciones proporcionadas. La programación competitiva es reconocida y respaldada por varias empresas multinacionales de software e Internet, como Google y Facebook. Hay varias organizaciones que organizan competiciones de programación de forma regular.
