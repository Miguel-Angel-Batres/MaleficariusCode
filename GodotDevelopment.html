Godot Engine


¿What is Godot Engine?

Godot Engine is an open-source, cross-platform game engine that is used for the development of 2D and 3D games, as well as interactive simulations and visualizations. It is free to use, and its source code is available under the MIT license.


¿Why I use Godot?:

There are so much reasons i like godot better than any other game engine:

  • Multi-platform Support: As it been said, Godot is cross-platform, so you can export your games to various platforms, including Windows, MacOS, Linux, iOS, HTML5, Android and more.
  • Scene System: Games are built using a scene system, this means that everything is a scene, for example: A bed is a scene inside a house, which is also a scene.
  • Accessibility: Godot is a very light game engine, and this is so good for those who dont have a good computer or dont have the money to buy a better machine
  • Cost Effectiveness: Godot is tottaly free, and the most important, You dont have to pay if your game was successful.
  • Extensibility:The engine is highly extensible, allowing developers to create custom modules, plugins, and tools to enhance its capabilities.
  • Community and Documentation: Godot has an active and growing community, and it offers comprehensive documentation and tutorials to help developers get started and solve problems.

Godot Engine is a popular choice among indie game developers and hobbyists due to its accessibility, cost-effectiveness, and the ability to export to multiple platforms. Its continual development and community support contribute to its widespread adoption in the game development community.

History of Godot

  • Origins (2007): Godot was created by Juan Linietsky, an Argentinian software developer, in 2007. Initially, it was developed as a proprietary engine for a client project.
  • Open Sourcing (2007 - 2008): In 2007, Linietsky decided to release the engine as an open-source project under the MIT license. This marked the beginning of Godot as a community-driven project.
  • Version 1.0 (April 2014): The first stable release, Godot 1.0, was launched in April 2014. It provided a range of features for 2D and basic 3D game development.
  • Version 2.0 (February 2016): Godot 2.0 was a major update, introducing significant improvements to the engine, including a new animation system, enhanced 3D capabilities, and a revamped scene system.
  • Version 3.0 (January 2018): Godot 3.0 was a landmark release that brought many new features and improvements, including a new Vulkan renderer, a redesigned physics engine, and GDNative for writing plugins in other languages.
  • Godot 4.0: This includes a new Vulkan-based renderer for improved graphics capabilities.

One of the most popular games made with Godot is Sonic Colors, a fan-made game that was developed by a team of dedicated fans. The game is a 2D platformer that features Sonic the Hedgehog and his friends as they embark on a new adventure to save the world from the evil Dr. Eggman. The game has received positive reviews from players and critics alike, and it showcases the capabilities of the Godot Engine for creating high-quality games.
